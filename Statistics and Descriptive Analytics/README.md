# Statistics and Descriptive Analytics

Projects building the statistical foundation for business analytics — from descriptive summaries and hypothesis testing through regression modeling and classification. The work emphasizes **interpretability** and translating statistical results into business decisions.

---

## Projects

### Basic Statistics
Descriptive analytics applied to a real baseball dataset (Moneyball context) — distributions, summary statistics, and exploratory analysis.

| File | Description |
|---|---|
| `Basic Statistics.ipynb` | Descriptive statistics walkthrough |
| `Baseball.csv` | Dataset |
| `Moneyball dataset description.docx` | Data dictionary |

### Linear Regression
Builds and interprets linear regression models to predict continuous outcomes. Applied to diamond pricing — modeling how cut, color, clarity, and carat drive price.

| File | Description |
|---|---|
| `Linear Regression.ipynb` | Regression modeling and interpretation |
| `diamond.csv` | Dataset |
| `R_ Pricing the C's of Diamond Stones.pdf` | Problem brief |

### Logistic Regression
Applies logistic regression for binary classification — predicting outcomes like spam detection and survival. Covers model evaluation with confusion matrix, accuracy, recall, and F1-score.

| File | Description |
|---|---|
| `Logistical Regression.ipynb` | Core logistic regression analysis |
| `Logistical Regression Challenge.ipynb` | Extended challenge |
| `spam.csv` / `titanic.csv` | Datasets |
| `R_ Spam E-mail Data.pdf` / `Logistic Regression Challenge.pdf` | Problem briefs |

## Skills Demonstrated

- Descriptive statistics and distribution analysis
- Linear and multilinear regression modeling
- Logistic regression for classification
- Model evaluation (R², AUC, confusion matrix, F1-score)
- Feature interpretation and business storytelling
