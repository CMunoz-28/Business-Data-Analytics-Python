# Segmentation

Projects applying customer segmentation techniques to identify distinct behavioral groups within a customer base. The work combines **unsupervised machine learning** with **business interpretation** to turn raw data into actionable audience segments.

---

## Projects

### RFM Analysis
Segments customers using **Recency, Frequency, and Monetary** value — a classic framework for ranking customer engagement and value. Identifies high-value, at-risk, and dormant segments for targeted marketing.

| File | Description |
|---|---|
| `RFM.ipynb` | Core RFM segmentation analysis |
| `RFM Challenge.ipynb` | Extended challenge with full segmentation and business recommendations |

### Gaussian Mixture Model (GMM)
Applies **probabilistic clustering** via Gaussian Mixture Models to a credit card customer dataset. Unlike hard clustering (k-means), GMM assigns soft probabilities to each segment — capturing the natural ambiguity in customer behavior.

| File | Description |
|---|---|
| `Gaussian Mixture Model.ipynb` | Core GMM exercise |
| `Gaussian Mixture Model Challenge.ipynb` | Extended challenge with full clustering analysis |
| `Clustering.ipynb` | Broader clustering analysis and comparison |
| `Clustering Challenge.ipynb` | Extended clustering challenge |
| `Gaussian Mixture Model Challenge.pdf` | Challenge brief |
| `CC GENERAL.csv` / `CreditCard.csv` | Credit card customer datasets |
| `CC GENERAL data dictionary.docx` | Variable definitions |
