# Econometrics and Causal Inference

Projects applying econometric methods and causal inference techniques to real-world business questions. The work goes beyond correlation to estimate the actual **causal effect** of interventions — marketing campaigns, policy changes, platform updates — using rigorous statistical frameworks.

---

## Projects

### Google Causal Impact
Applies **Google's Causal Impact** library (Bayesian structural time-series) to measure the incremental effect of a marketing intervention on a key outcome metric. Isolates true lift from seasonal trends and confounding variables.

| File | Description |
|---|---|
| `Google Causal Impact.ipynb` | Core exercise — method walkthrough and application |
| `Google Causal Impact Challenge.ipynb` | Extended challenge with full analysis and interpretation |
| `CausalImpact.ipynb` | Additional applied analysis |
| `Google Causal Impact Challenge.pdf` | Challenge brief |

### Matching
Uses **propensity score matching** to create comparable control and treatment groups from observational data — enabling causal inference without a randomized experiment.

| File | Description |
|---|---|
| `Matching.ipynb` | Core exercise — matching methodology and application |
| `Matching Challenge.ipynb` | Extended challenge with full analysis |
| `Matching Challenge.pdf` | Challenge brief |
| `school.csv` / `stackoverflow.csv` | Datasets used in analysis |
